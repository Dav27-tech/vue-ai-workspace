# AI Workspace

A modern dashboard application built with **Vue.js 3**, **TypeScript**,
and **Tailwind CSS**. The project demonstrates how to create responsive
user interfaces using reusable Vue components while following modern
frontend development practices.

The application presents a centralized workspace where users can explore
AI tools, view project statistics, and browse recent conversations.
Although the data is static, the project is structured similarly to a
real-world application, making it an excellent foundation for learning
Vue.js component architecture and TypeScript.

---

## Features

- Responsive dashboard layout
- Modern navigation bar
- Reusable Vue components
- AI tools dashboard
- Project statistics cards
- Recent conversations list
- TypeScript interfaces
- Tailwind CSS utility-first styling
- Mobile-first responsive design

---

## Technologies

- Vue.js 3
- TypeScript
- Vite
- Tailwind CSS

---

## Project Structure

```text
src/

├── assets/

│   ├── Statistics.vue
│   ├── ToolCard.vue
│   ├── ToolGrid.vue
```

---

## Learning Objectives

This project is designed to help students practice:

- Creating Vue projects with Vite
- Working with Vue Single File Components
- Using TypeScript in Vue applications
- Building reusable components
- Passing data with props
- Rendering lists using `v-for`
- Organizing project data into modules
- Building responsive layouts with Tailwind CSS
- Applying the utility-first CSS philosophy

---

## Getting Started

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

---

## Application Overview

The dashboard is composed of several reusable components:

- **Header** – Responsive navigation bar with search and user profile
- **Welcome** – Introduction section
- **Statistics** – Dashboard summary cards
- **ToolGrid** – Collection of AI assistant cards
- **ConversationList** – Recent conversations
- **Footer** – Application footer

Each component is designed to demonstrate Vue's component-based
architecture while using Tailwind CSS for styling.
